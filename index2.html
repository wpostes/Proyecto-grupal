HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Permite definir la estructura y el contenido de una página web mediante etiquetas y elementos.
Para crear una lista en HTML, puedes usar las etiquetas `
    ` para una lista no ordenada o `
      ` para una lista ordenada. Luego, utiliza `
    1. ` para cada elemento de la lista.
